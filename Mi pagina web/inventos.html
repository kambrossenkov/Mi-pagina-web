Mi objetivo es acercarles a estos comerciantes una solución personalizada según su necesidad concreta en ese momento.

Mi primera página web

Esta es mi primera página web que desarrollé para la materia Programación Web el 11/21/22, la cual me permitió incursionar en los lenguajes de programación y ver soluciones tecnológicas para negocios. Acá se verá un caso de un negocio de venta de alcohol para que la gente pueda pedirla a través de una web, destacándose que es responsive para desktop y celular. Vale recordar el contexto: no existía IA para resolver los problemas que surgían — había que interpretar y saber encontrar la mejor solución por uno mismo.
